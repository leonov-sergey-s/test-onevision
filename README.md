## How to run application

#### Tests

```shell script
./gradlew test
```

#### Build

```shell script
./gradlew clean build
```

#### Run spring boot application

```shell script
./gradlew bootRun
```

Open http://localhost:8080/swagger-ui/index.html in your browser to view application api.
